¿Qué es el lenguaje HTML?

HTML son las siglas de HyperText Markup Language
HTML es el lenguaje estandar para crear páginas web, con el se puede describir una estructura de una página web.
HTML consta de una serie de elemntos que le dicen al navegador com debe comportarse

Ejemplo:

<!DOCTYPE html
<html >
<title > Tutorial de HTML </title >
<body >

<h1 > This is a heading </h1 >
</body >
</html >
